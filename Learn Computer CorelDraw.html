Corel Draw

Corel-Draw is one of the most popular and powerful graphics programs and gives designers a most rewarding and enjoyable work experience. It is built and designed to meet the day to day demands of working designers.

Corel-Draw is a graphic enhancement program that provides a eye quality graphic tool which is both popular and easy to use the tools enables gratifying results, this is the most popular graphic package which allow users to create logos, posters, greeting card, business card, letter heads, stickers, fliers etc.

CorelDRAW is an intuitive and versatile graphics application for creating high-quality vector illustrations, logo designs, and page layouts.

Opening CorelDraw

  • Switch on the computer and take the following steps:
  • Go to Start and select Programs from the list. (The welcome screen at start-up is shown in the next figure).
  • Find the Corel section and select CorelDraw.
  • Choose Open Graphic for old design. New Graphic creates a new, one page document. Template allows access to professionally created templates from the library. Corel Tutor gives a brief overview of how to use the program.

Features Of Corel-Draw

  • Title Bar

    This is the first horizontal bar on the top of the window environment. It carriers the name of the name of the application you are currently running and also the name of the document which you are actively working on. It contains the minimize, restore, maximize and close button.

    The Title Bar

  • Menu Bar

    The menu bar is located below the title bar. It contain command jacket from files to help.

    The Menu Bar

  • Standard Tool Bar

    This is located below the menu bar and contain some important shop keys of the menu bar that standardize your documents.

    The Standard Tool Bar

  • Property Tool Bar

    This is located below the standard tool bar. It display all information about any command selected. It also provides you with more features on a command selected.

    The Property Tool Bar

  • Ruler Bar

    It is usually vertical and horizontal ruler bars. They are used for maintaining measurements unlike in columns.

    The Horizontal and Vertical Ruler bars

  • Tool Bar

    The tool bar consists of several tools which are used to perform graphical operatives. This include the following:

    1. Pick Tool

      The pick tool selects objects you can use it to move or stretch, scale, rotate and skew objects.

    2. Shape Tool

      Shape tools reshape objects by moving nodes, lines, and control points.

    3. Crop

      This tool is used to cut objects or image within the environment.

    4. Zoom Tool

      It is used to Zoom in and Zoom out within the interface.

    5. Freehand Tool

      Freehand tools draw lines and curves and can also be used to trace bitmap.

    6. Artistic Media

      Add artistic brush, spray, and calligraphic effects by using freehand strokes.

    7. Polygon

      It is used for drawing Polygons and it also consists of several tools such as; Star, Impact Tool, Basic shapes.

    8. Interactive Fill

      adds another color to any object.

    9. Fill Tool

      assigns the fill of any object.

    10. Square Tool

      draws rectangles and squares.

    11. Circle Tool

      draws ellipses and circles.

    12. Basic Shapes Tool

      a collection of objects, which you can add to a drawing.

    13. Text Tool

      adds either artistic or paragraph text to your drawing.

    14. Eyedropper Tool

      allows you to select a color within objects, especially a bitmap, and allows to apply that color to another object.

    15. Interactive Blend Tool

      Allows you to Merge objects

    16. Drawing Window

      The area outside the drawing page boarded by the scroll bars and application controls.

      A CorelDraw drawing page
    17. Drawing Page

      The rectangular area inside the drawing window. It is the printable area of your work area.

    18. Color Palette

      It is used for choosing the color of your choice on your drawing.

      The Color Palette
