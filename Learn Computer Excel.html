Specifying a Range of Cells

A range is defined as a block of cells. It maybe a single cell, group of adjacent cells or the entire worksheet. Range of cells is specified by highlighting a group of cells or typing the addressed of the range in the name box located in the formatting tool bar. To specify a range of cells: By highlighting adjacent cells

  • Method 1:- Click on the cell in the top left corner of the range and drag to the bottom-right corner of the range.
  • Method 2:- Move the cursor to the top left corner of the range and hold down the shift key and use the arrow keys to move the bottom-right corner of the range.
By Typing range address this includes:-
  • Click inside the name box and type the range address e.g. B3:F14
  • Press the (Enter) key.

Functions

MS-Excel has many built functions to perform calculations such as summations, logarithm, average, and square root. Like formulas, functions must begin with an equal (=) sign followed by the function name and range of cells in pair of brackets. The range of cells in pair of brackets is called the argument. E.g. =SUM(B6:D6) sums all the values in the range of B6:D6 sum-function name (B6:D6)-Argument =AVG(G10:H15) calculates the average of the values in the range G10:H15 =MAX(A1:C7) finds the maximum value in the range A1:C17 The expression =SUM(B6:D6) is equivalent to =B6+C6+D6 some MS-Excel functions are listed below MS-Excel Built In Functions Function Description Example =SUM Sum all the value in a range =SUM(B1:F10) =AVG Calculates the average of a range =AVG (G10:H15) =MAX Find the largest value in a range =MAX (A1:C7) =SQRT Calculates the square root of a number =SQRT(B16) =MIN Finds the smallest value in a range =MIN(D1:F10) =LOG Find the log (base 10 of a number) =LOG(B16) To enter a function, this includes:- Place the cursor where you want to the result to appear, type in the function required. The function must begin with an equal sign (=) followed by the function name and then the range of cells in pairs of brackets. Using Auto-Sum To Calculate Totals Auto-Sum is used to calculate the sum of values in rows and columns. It cannot perform calculations other than summations. Ʃ the Auto-Sum button.

COPYING FORMULAS, FUNCTION AND SUMS TO OTHER CELLS.

In practical the =Sum function was typed for each time to find the total scores. This procedure is time consuming if you were to find the scores for many times. This problems can be overcome by copying the formula or function to other cells. Formula or function is copied using one of the following methods.

  • Copy and Paste commands in the edit menu
  • Copy and Paste buttons on the total bar: or
  • Fill handle (Auto fill) in the right-corner of the cursor
to copy a sum, formula or function. This includes:
  • click on the cell containing the sum, formula or function
  • click on the copy button on the total bar. (or selection the copy command from the edit menu) Select the range of cells to fill with the sum, formula or function.
  • click on the Paste button on the tool bar. (or select the Paste command from the edit menu)
