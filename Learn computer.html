INTRODUCTION TO COMPUTER

A computer is an electronic device, which accepts and stores data input, process the data input, and generates the output in a required format. A computer system has the three main components; HARDWARE, SOFTWARE, and PEOPLEWARE.

A Computer

Components of the Computer

The computer consists of two major components, they are HARDWARE and SOFTWARE. This part works together for effective operation of the computer system for the facts, that none of the component can function without others.

Computer Software

Computer software is a set of programs and associated documentation and data. This is in contrast to hardware, from which system is built and which actually performs the work examples: FireFox, Google Chrome, CorelDRAW etc.

There are two types of software

  • System Software
  • Application Software

System Software

The system software is a collection of programs designed to operate, control, and extend the processing capabilities of the computer itself. System software is generally prepared by the computer manufacturers. These software products compromise of programs written in low-level languages, which interact with the hardware at a very basic level. System software serves as the interface between the hardware and the end users.

Some examples of system software are Operating System, Compliers, Interpreter, Assemblers, etc.

Here is a list of some of the most important features of a system software

  • Close to the system
  • Fast in speed
  • Difficult to design
  • Less Interactive
  • Smaller in size
  • Difficult to manipulate
  • Generally written in low-level language

Application Software

Application software are designed to satisfy a particular need of a particular environment. All software applications prepared in a computer lab can come under the category of Application software.

The following are the examples of Application software

  • Microsoft Word
  • Microsoft Excel
  • Microsoft PowerPoint
  • CorelDraw

Analog Computer

An analog computer is a form of computer that uses continuous physical phenomena such as electrical, mechanical, or hydraulic quantities to model the problem being solved.

Digital Computer

A digital computer is a computer that performs calculations and logical operations with quantities represented as digits, usually in the binary number system.

Hybrid Computer (Analog + Digital)

A hybrid computer is a computer that can combine the functionality of both analog and digital computer. It is capable of handling input and output in both digital and analog signals. A hybrid computer system setup offers a cost effective method of performing complex simulations.

Generations of Computer

  1. 1st Generation (1940-1956) Vaccum Tubes:

    They are use tubes as gravity and magnetic drums for memory. They are big and contain lots of space. They are expensive to run. These computer were limited to solving one problem at a time. They relied machine language.

  2. 2nd Generation (1956-1963) Transistors

    They use transistors are circuity instead of Vaccum Tubes. They are smaller, faster, cheaper and less heavily on electricity use. They use symbolic languages which helps in creating instructions in words.

  3. 3rd Generation (1964-1991) Intergrated

    Circuits: These were the first computers where users intergrated using keyboard and monitor which interface with the operating system. This helped the machines to run several operations at once using a central program. These machines are cheaper and are called semi-conductors.

  4. 4th Generation (1992-2010) Micro-Processors

    The fourth generation gave birth to the computer component like:

    1. Central Processing Unit (CPU)
    2. Memory
    3. Output Controls
    4. Input Controls

    This is the beginning of revolution for internet.

  5. 5th Generation (2010-Till Date)

    Some of these devices are remaining to merge such as voice recognitions. The main reason for this is to create machines that can process and respond natural languages and have the capacity to learn and organize themselves.

Computer Netrworking

A computer network is a system in which multiple computers are connected to each other to share information and resources.

Characteristics of a Computer Network

  • Share resources from one computer to another
  • Create files and store them in one computer, access those files from the other computer(s) connected over the network.
  • Connect a printer, scanner, or a fax machine to one computer within the network and let other computers of the network use the machines available over the network
Following is the list of hardware's required to set up a computer network.
  • Network Cables
  • Distributors
  • Routers
  • Internal Network Cards
  • External Network Cards

Network Cables

Network cables are used to connect computers. The most commonly used cable is Category 5 cable RJ-45

Distributors

A computer can be connected to another one via a serial port but if we need to connect many computers to produce a network, this serial connection will not work.

The solution is to use a central body to which another computers, printers, scanners, etc. can be connected and then this body will manage or distribute network traffic

TERMINOLOGIES OF WORD PROCESSING OPERATIONS

  1. Formatting Document

    The ability of the word processing program to show the user how the document will appear on paper after printing for the user to confirm his initial desire. That is, setting the margins top, bottom, left and right margins, choosing the paper size e.g (A4, Letter) indicating page numbering, select font size, text justification etc. Therefore, enhances the appearance and readability of the word processed document when printed.

  2. Editing

    The process of creating wrongly typed words, it can also be the deletion of a small character, a complete word or even several lines and then word processing program adjusts the text to move back to the spaces previously occupied to the deleted text.

  3. Word Wrap

    The process whereby the word programs automatically determines the maximum number of characters that can accommodate in a line and transfers control to the beginning of a new line for continuation. It can also be explained as a common feature of word processors which extends the defined right margin, is automatically wrapped to the next line. It will continue to the end of each line. The enter key is pressed only to start a new paragraph.

  4. Scrolling

    The moving up or down the screen while displaying the contents of the document.

  5. Find and Replace

    The ability of the word processing program to allow a user replace a single word or phrase anywhere it occurs in the document with the desired word or phrase.

  6. Text Justification

    The alignment of text against a margin either left, right, center or full alignment. That is, alignment to the left is called left justification (Ctrl L), right justification (Ctrl R), centered text is called centre justification (Ctrl E), while a combination of left and right is called full justification (Ctrl J).

  7. Spelling and Grammar

    Any wrongly typed words which are not in the computer dictionary will be underlined with wary red line. As you type, word automatically check your document and underline possible spelling and grammatical errors. To correct an error display a shortcut menu by right clicking on the misspell word and then select (click) the correction you want. During typing words uses wavy red underlines to indicate possible grammatical errors, then click the correction you want on the shortcut menu.

  8. Copy

    Copy is a command used to duplicate a file or folder while the original remains in its initial position. Copy Ctrl+C

  9. Paste

    The word paste command is a result of action copied. Files or folders which enable the user to paste it in order to duplicate the file or folder copied. Paste Ctrl+V .
